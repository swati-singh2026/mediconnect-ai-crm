# 🩺 MediConnect AI CRM

AI-first Healthcare Professional (HCP) CRM platform powered by React, FastAPI, LangGraph, and Groq LLM.

## Tech Stack

- React
- Redux Toolkit
- FastAPI
- LangGraph
- Groq (Gemma2-9B-IT)
- PostgreSQL

## Modules

- Log Interaction
- AI Chat
- HCP Management
- Interaction Timeline
- AI Summary
